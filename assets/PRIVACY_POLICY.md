# Privacy Policy for Rich Link Copier

**Last Updated:** January 6, 2026

## Overview

Rich Link Copier is a Chrome extension that helps you copy rich-formatted links from web pages.

## Data Collection

**We do not collect, store, or transmit any personal data.**

This extension:
- ✅ Operates entirely locally in your browser
- ✅ Does not send any data to external servers
- ✅ Does not track your browsing activity
- ✅ Does not collect analytics or telemetry
- ✅ Does not use cookies or local storage for tracking

## Permissions Explained

The extension requires the following permissions:

### activeTab
- **Why:** To read the current page's title and URL when you press the keyboard shortcut or click the extension icon
- **Usage:** Only activated when you explicitly trigger the extension
- **Data:** Page title and URL are read temporarily and never stored or transmitted

### scripting
- **Why:** To inject content scripts that show notifications on the page
- **Usage:** Injects notification UI code when you copy a link
- **Data:** No data is collected

### clipboardWrite
- **Why:** To copy formatted links to your clipboard
- **Usage:** Only when you explicitly request to copy a link
- **Data:** Link data is written to your system clipboard (controlled by your OS)

### host_permissions: <all_urls>
- **Why:** To work on any website you visit
- **Usage:** Extension only activates when you trigger it
- **Data:** No data is collected or transmitted

## Third-Party Services

This extension does not use any third-party services, analytics, or tracking tools.
