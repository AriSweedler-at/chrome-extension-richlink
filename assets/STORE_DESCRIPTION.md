# Chrome Web Store Listing

## Short Description (132 characters max)

Copy rich-formatted links from any webpage with a keyboard shortcut or popup menu. Supports Google Docs, GitHub, and more.

## Detailed Description

Rich Link Copier makes it easy to copy properly formatted links from web pages. Press Command+Shift+C (or Ctrl+Shift+C on Windows/Linux) to instantly copy a rich link, or click the extension icon to choose from multiple format options.

### Features

**Universal Support**
Works on any website with smart handlers for:
- Google Docs (with section headings)
- GitHub Pull Requests
- Atlassian Confluence
- Airtable
- Spinnaker
- Any other website (fallback)

**Multiple Copy Formats**
Cycle through different formats for each page:
- Page title with link
- Section/header links
- Raw URL (plain text only)

**Two Ways to Use**
1. **Keyboard Shortcut:** Press Command+Shift+C repeatedly to cycle through formats
2. **Popup Menu:** Click the extension icon to see all available formats

**Smart Clipboard**
Copies both HTML and plain text formats:
- Paste in Slack/Gmail → clickable link
- Paste in plain text editor → "Title (URL)" format
- Raw URL option → just the URL, no formatting

**Toast Notifications**
Visual feedback shows which format was copied with clear labels like:
- "Copied to clipboard [2/4]"
- "Header: Introduction"

### Privacy

This extension:
- Does NOT collect any data
- Does NOT track your browsing
- Works entirely offline
- Is open source

### How to Use

1. Navigate to any webpage
2. Press **Command+Shift+C** (Mac) or **Ctrl+Shift+C** (Windows/Linux)
3. The rich link is copied to your clipboard
4. Press again to cycle to the next format (with header, raw URL, etc.)

OR

1. Click the extension icon
2. See all available formats for the current page
3. Click any format to copy it

### Supported Sites

Special handlers provide enhanced functionality for:
- **Google Docs** - Copy with document title or specific heading
- **GitHub** - Copy PR title and link
- **Atlassian Confluence** - Copy page title
- **Airtable** - Copy record title from specific bases
- **Spinnaker** - Copy pipeline and application names
